# Ver la tecla predeterminada de cualquier acción

La tabla de acciones del diálogo Keyboard Shortcuts muestra la tecla predeterminada de cada acción junto a su asignación actual, de modo que puede consultar cuál era el atajo original sin modificar nada.

## Antes de comenzar

- No se requiere conexión a una radio.

## Pasos

1. Haga clic en `View > Configure Shortcuts...` para abrir el diálogo Keyboard Shortcuts.
2. En la tabla de acciones, ubique la acción que desea consultar. Cada fila muestra cuatro columnas: **Action**, **Category**, **Current Key** y **Default Key**.
3. Lea el valor en la columna **Default Key** correspondiente a esa acción.

Para reducir la lista, escriba parte del nombre de la acción en el campo `Filter:`, o seleccione una categoría en el cuadro combinado `Category:` situado sobre la tabla.

## Qué hace cada control

| Control | Tipo | Comportamiento |
|---|---|---|
| `Filter:` | Campo de texto | Filtra la tabla de acciones para mostrar las filas cuyo texto de Action o Category coincida con lo que escribe. |
| `Category:` | Cuadro combinado | Filtra la tabla de acciones para mostrar una única categoría. El valor predeterminado es **All**. |
| Action table | Lista | Muestra todas las acciones con las columnas: Action, Category, Current Key, Default Key. |

## Sugerencias

- La columna **Default Key** siempre refleja el valor predeterminado de fábrica, incluso después de haber reasignado o eliminado la acción. Úsela como referencia antes de decidir si desea restaurar una asignación.
- Para restaurar una sola acción al valor que aparece en **Default Key**, seleccione su fila y haga clic en `Reset to Default`. Consulte [Restablecer un atajo a su tecla predeterminada](reset-one-shortcut-to-its-default-key.md).
