# Configurar modos digitales paso a paso

Esta página explica cómo abrir la guía de configuración de modos digitales incluida en AetherSDR. La guía lo lleva a través de la configuración de modos de datos (como FT8, JS8Call u otros modos digitales) mediante el sistema de ayuda sin conexión de AetherSDR.

## Antes de comenzar

- AetherSDR debe estar instalado. No se requiere conexión a una radio para leer esta guía.

## Pasos

1. Haga clic en **Help** en la barra de menú.
2. Haga clic en **Configuring Data Modes...**.
3. Se abre el HelpDialog con el título correspondiente al tema de modos de datos. Lea el visor de Markdown para obtener instrucciones de configuración paso a paso.
4. Cuando termine, haga clic en **Close**.

## Consejos

- La nota al pie indica: "The Help menu keeps each guide separate so you can reopen just the topic you need." Cada paso de modos digitales en la guía es independiente — puede cerrar y volver a abrir el diálogo en cualquier momento sin perder la configuración de su radio.
- El diálogo es redimensionable. Arrastre los bordes para tener más espacio de lectura si el tamaño predeterminado de 760 × 680 es demasiado pequeño para su pantalla.
- Este tema de ayuda está disponible sin conexión. No se requiere conexión a internet.

## Solución de problemas

- **El visor de Markdown muestra "Help file not available"** — El archivo de ayuda incluido falta en su instalación. Reinstale AetherSDR para restaurar el archivo.
