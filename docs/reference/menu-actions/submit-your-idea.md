# Enviar su idea... 💡

Esta página explica cómo usar `Help > Submit your idea... 💡` para reportar un error o proponer una nueva función mediante un asistente de inteligencia artificial. El flujo de trabajo copia un prompt predefinido en el portapapeles y abre el sitio web del proveedor de IA para que el asistente le ayude a verificar duplicados y estructurar su envío.

## Antes de comenzar

- Necesita una conexión a internet activa. El ítem del menú consulta GitHub para verificar si existe una versión más reciente de AetherSDR antes de abrir el diálogo.
- Necesita una cuenta con al menos uno de los proveedores de IA compatibles: Claude, ChatGPT, Gemini, Grok o Perplexity.

## Pasos

1. Haga clic en `Help > Submit your idea... 💡`.
2. Si AetherSDR detecta que hay una versión más reciente disponible, se muestra una advertencia. Revísela antes de continuar, ya que su problema podría estar resuelto en la versión más reciente.
3. Se abre el diálogo AI-Assisted Issue Reporter. Lea las instrucciones paso a paso que se muestran en el diálogo.
4. Haga clic en el botón del proveedor de IA de su preferencia: **Claude**, **ChatGPT**, **Gemini**, **Grok** o **Perplexity**.
5. AetherSDR copia un prompt predefinido en el portapapeles y abre el sitio web del proveedor seleccionado en su navegador predeterminado.
6. En el navegador, pegue el contenido del portapapeles en el campo de chat del asistente de IA y envíelo.
7. El asistente de IA consulta el repositorio de GitHub de AetherSDR en busca de problemas duplicados y genera un reporte de error o solicitud de función estructurado. Revise el resultado del asistente y utilícelo para abrir un issue en GitHub.

## Qué hace cada control

| Botón | Acción |
|---|---|
| **Claude** | Copia el prompt predefinido en el portapapeles y abre el sitio web de Claude. |
| **ChatGPT** | Copia el prompt predefinido en el portapapeles y abre el sitio web de ChatGPT. |
| **Gemini** | Copia el prompt predefinido en el portapapeles y abre el sitio web de Gemini. |
| **Grok** | Copia el prompt predefinido en el portapapeles y abre el sitio web de Grok. |
| **Perplexity** | Copia el prompt predefinido en el portapapeles y abre el sitio web de Perplexity. |

## Consejos

- El prompt indica al asistente de IA que busque issues existentes en GitHub antes de redactar uno nuevo. Permita que el asistente complete ese paso antes de copiar su resultado en un nuevo issue.
- Si aparece la advertencia de versión, considere actualizar AetherSDR primero y verificar si el error o la función faltante ya ha sido corregida.
