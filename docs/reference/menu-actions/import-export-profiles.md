# Importar/Exportar Perfiles

Esta página cubre el elemento de menú `Profiles > Import/Export Profiles...`, el cual está diseñado para realizar copias de seguridad y compartir perfiles de AetherSDR.

## Antes de comenzar

- Esta funcionalidad aún no está implementada. El elemento de menú existe, pero no abre un cuadro de diálogo funcional.

## Pasos

1. Haga clic en `Profiles > Import/Export Profiles...`.
2. No se abre ningún cuadro de diálogo. La funcionalidad aún no está implementada.
