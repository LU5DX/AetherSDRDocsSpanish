# Ayuda de AetherSDR

Abre la documentación principal del usuario de AetherSDR. Utilice esta opción cuando necesite consultar el manual completo desde la aplicación.

## Antes de comenzar

- AetherSDR debe estar en ejecución.

## Pasos

1. Haga clic en **Help** en la barra de menú.
2. Haga clic en **AetherSDR Help...**.

Se abrirá la documentación principal del usuario.
