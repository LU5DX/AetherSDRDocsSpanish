# Cables USB

`Settings > USB Cables...` abre la pestaña USB Cables del diálogo Radio Setup, donde se asignan las conexiones de cable USB para el control CAT y el enrutamiento de audio.

## Antes de comenzar

- AetherSDR debe estar conectado a una radio Flex.
- El cable USB debe estar físicamente conectado entre la radio y su computadora antes de asignarlo aquí.

## Pasos

1. Haga clic en `Settings > USB Cables...`.
2. El diálogo Radio Setup se abre en la pestaña USB Cables.
3. Configure las asignaciones de CAT y audio para cada cable USB conectado utilizando los controles de la pestaña.
4. Cierre el diálogo cuando termine.
