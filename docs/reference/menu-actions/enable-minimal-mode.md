# Habilitar el modo minimalista

El modo minimalista oculta la barra de título y los controles para dar al panadapter la máxima área de pantalla disponible. Úselo cuando desee una vista limpia y despejada del espectro.

## Antes de comenzar

- AetherSDR debe estar en ejecución y conectado a una radio FLEX-8600.

## Pasos

1. Haga clic en `View` en la barra de menú.
2. Haga clic en `Minimal Mode`.

El elemento es seleccionable. Una marca de verificación junto a `Minimal Mode` indica que está activo. Si hace clic nuevamente, se elimina la marca de verificación y se restauran la barra de título y los controles.

También puede activar o desactivar el modo minimalista en cualquier momento con el atajo de teclado **Ctrl+M**.

## Consejos

- Si también desea eliminar el marco de la ventana por completo, consulte [Habilitar ventana sin marco](enable-frameless-window.md).
- Para recuperar los controles ocultos cuando la barra de menú no está visible, presione **Ctrl+M** para desactivar el modo minimalista.
