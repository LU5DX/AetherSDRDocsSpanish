# Activar el Modo Mínimo

El Modo Mínimo oculta la barra de título y los controles para otorgar al panadapter la mayor área de pantalla disponible. Úselo cuando desee una vista limpia y despejada del espectro.

## Antes de comenzar

- AetherSDR debe estar en ejecución y conectado a una radio FLEX-8600.

## Pasos

1. Haga clic en `View` en la barra de menú.
2. Haga clic en `Minimal Mode`.

El elemento es seleccionable. Una marca de verificación junto a `Minimal Mode` indica que está activo. Al hacer clic nuevamente se elimina la marca y se restauran la barra de título y los controles.

También puede activar o desactivar el Modo Mínimo en cualquier momento con el atajo de teclado **Ctrl+M**.

## Consejos

- Si también desea eliminar completamente el marco de la ventana, consulte [Activar la ventana sin marco](enable-frameless-window.md).
- Para recuperar los controles ocultos cuando la barra de menú no es visible, presione **Ctrl+M** para desactivar el Modo Mínimo.
