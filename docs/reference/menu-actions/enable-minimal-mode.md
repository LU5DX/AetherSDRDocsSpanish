# Activar el Modo Mínimo

El Modo Mínimo oculta la barra de título y los controles para maximizar el área del panadapter. Úselo cuando desee la vista más amplia posible de la pantalla de espectro.

## Antes de comenzar

- AetherSDR debe estar en ejecución y conectado a una radio.

## Pasos

1. Haga clic en `View` en la barra de menú.
2. Haga clic en `Minimal Mode`.

El elemento de menú es seleccionable. Haga clic en él nuevamente para restaurar la barra de título y los controles.

## Consejos

- El atajo de teclado `Ctrl+M` activa y desactiva el Modo Mínimo sin abrir el menú.
- Para también eliminar las decoraciones de ventana del sistema operativo, active `View > Frameless Window` (`Ctrl+Shift+F`) junto con el Modo Mínimo.
