# Habilitar atajos de teclado

`View > Keyboard Shortcuts` es un elemento de menú con marca de verificación que habilita o deshabilita el procesamiento de atajos de teclado en todo AetherSDR. Cuando está habilitado, los atajos de teclado —como el uso de las teclas de flecha para ajustar valores de controles deslizantes— se activan.

## Antes de comenzar

- AetherSDR debe estar en ejecución y conectado a una radio FLEX-8600.

## Pasos

1. Haga clic en `View` en la barra de menú.
2. Haga clic en `Keyboard Shortcuts` para activar o desactivar la opción.
   - Una marca de verificación junto a la etiqueta indica que los atajos de teclado están habilitados.
   - La ausencia de marca de verificación indica que los atajos de teclado están deshabilitados.

## Consejos

- Si las teclas de flecha u otros controles de teclado dejan de responder en controles deslizantes u otros elementos, verifique que `View > Keyboard Shortcuts` esté marcado.
- Para personalizar qué teclas están asignadas a qué acciones, use `View > Configure Shortcuts...`, que abre el editor de atajos de teclado.
