# Configuración de los controles de AetherSDR

Esta página describe qué hace el elemento de menú `Help > Configuring AetherSDR Controls...`. Abre una guía dentro de la aplicación que cubre cómo personalizar applets, atajos de teclado y asignaciones MIDI.

## Antes de comenzar

- AetherSDR debe estar ejecutándose y conectado a una radio FLEX-8600, o como mínimo, iniciado hasta la ventana principal.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Configuring AetherSDR Controls...`.

Se abre la guía, que describe cómo personalizar el panel de applets, configurar atajos de teclado y establecer asignaciones de controladores MIDI.
