# Contribuir a AetherSDR

Esta página describe cómo abrir la guía de contribución, que explica cómo enviar código, informes de errores y solicitudes de nuevas funciones para AetherSDR.

## Antes de comenzar

- AetherSDR debe estar en ejecución.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Contributing to AetherSDR...`.

Se abre la guía de contribución. Siga las instrucciones que aparecen allí para enviar contribuciones de código, registrar informes de errores o proponer solicitudes de nuevas funciones.
