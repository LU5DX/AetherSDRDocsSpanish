# Cómo contribuir a AetherSDR

Esta página describe cómo abrir la guía de contribución, que cubre el envío de código, informes de errores y solicitudes de funciones para AetherSDR.

## Antes de empezar

- AetherSDR debe estar en ejecución.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Contributing to AetherSDR...`.

Se abrirá la guía de contribución. Siga las instrucciones allí para enviar contribuciones de código, reportar errores o proponer solicitudes de funciones.
