# Configuración de Modos de Datos

Esta página abre la guía integrada para configurar la operación en modos digitales en su radio Flex con AetherSDR.

## Antes de comenzar

- AetherSDR debe estar instalado y en ejecución.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Configuring Data Modes...`.

La guía se abre en el visor de ayuda de AetherSDR.
