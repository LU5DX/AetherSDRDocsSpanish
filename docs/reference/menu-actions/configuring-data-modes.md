# Configuración de modos de datos

Esta página describe cómo abrir la guía de configuración de modos de datos en AetherSDR. La guía cubre la configuración de AetherSDR para la operación en modos digitales como FT8, JS8 u otros modos de datos que requieren enrutamiento de audio o control CAT.

## Antes de comenzar

- AetherSDR debe estar en ejecución.
- Debe estar conectado a una radio FLEX-8600, o saber qué interfaz (DAX, TCI, CAT) desea configurar para su software de modos digitales.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Configuring Data Modes...`.

La guía de configuración de modos de datos se abre.
