# Ordenar la tabla de memorias por encabezado de columna

Haga clic en un encabezado de columna en el diálogo Memory Channels para ordenar la tabla por esa columna. Esto permite encontrar memorias rápidamente por frecuencia, nombre, grupo u otros campos.

## Antes de comenzar

- AetherSDR debe estar conectado al radio. El diálogo Memory Channels requiere una conexión de radio activa.
- Abra el diálogo Memory Channels mediante `Settings > Memory...`.

## Pasos

1. Abra `Settings > Memory...`.
2. En la tabla de memorias, haga clic en cualquier encabezado de columna ordenable — por ejemplo, **Frequency** o **Name**.
3. La tabla se ordena en forma ascendente. Aparece una flecha indicadora de orden en el encabezado.
4. Haga clic en el mismo encabezado nuevamente para invertir el orden a descendente.
5. Haga clic en un encabezado diferente para ordenar por esa columna, comenzando en orden ascendente.

## Consejos

- No todas las columnas son ordenables. Si al hacer clic en un encabezado no ocurre nada, esa columna no admite ordenamiento.
- La columna **Frequency** utiliza ordenamiento numérico, por lo que 14.225 se ubica correctamente entre 14.200 y 14.300, en lugar de tratarse como texto simple.
- El ordenamiento afecta únicamente el orden de visualización. No modifica el índice almacenado de ninguna memoria en el radio.
- Los filtros activos de búsqueda y perfil permanecen vigentes durante el ordenamiento. Para ver todas las memorias en el orden seleccionado, borre el campo **Search:** y establezca **Profile:** en su valor predeterminado.
