# Ordenar la tabla de memorias por el encabezado de columna

Haga clic en un encabezado de columna en el diálogo de Canales de Memoria para ordenar la tabla por esa columna. Esto le permite encontrar rápidamente memorias por frecuencia, nombre, grupo u otros campos.

## Antes de comenzar

- AetherSDR debe estar conectado a la radio. El diálogo de Canales de Memoria requiere una conexión activa con la radio.
- Abra el diálogo de Canales de Memoria a través de `Settings > Memory...`.

## Pasos

1. Abra `Settings > Memory...`.
2. En la tabla de memorias, haga clic en cualquier encabezado de columna que se pueda ordenar — por ejemplo, **Frequency** o **Name**.
3. La tabla se ordena en orden ascendente. Aparece una flecha indicadora de orden en el encabezado.
4. Vuelva a hacer clic en el mismo encabezado para invertir el orden a descendente.
5. Haga clic en un encabezado diferente para ordenar por esa columna, comenzando en orden ascendente.

## Consejos

- No todas las columnas se pueden ordenar. Si hacer clic en un encabezado no tiene efecto, esa columna no admite ordenamiento.
- La columna **Frequency** utiliza ordenamiento numérico, por lo que 14.225 se ordena correctamente entre 14.200 y 14.300, en lugar de hacerlo como texto plano.
- El ordenamiento solo afecta el orden de visualización. No cambia el índice almacenado de ninguna memoria en la radio.
- Los filtros de búsqueda y perfil activos permanecen vigentes durante el ordenamiento. Para ver todas las memorias en orden, limpie el campo **Search:** y establezca **Profile:** en su valor predeterminado.
