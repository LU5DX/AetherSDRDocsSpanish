# Ordenar la tabla de memorias haciendo clic en el encabezado de columna

Haga clic en un encabezado de columna del cuadro de diálogo Memory Channels para ordenar la tabla por esa columna. Esto le permite encontrar rápidamente memorias por frecuencia, nombre, grupo u otros campos.

## Antes de empezar

- AetherSDR debe estar conectado a la radio. El cuadro de diálogo Memory Channels requiere una conexión activa con la radio.
- Abra el cuadro de diálogo Memory Channels mediante `Settings > Memory...`.

## Pasos

1. Abra `Settings > Memory...`.
2. En la tabla de memorias, haga clic en cualquier encabezado de columna que se pueda ordenar — por ejemplo, **Frequency** o **Name**.
3. La tabla se ordena en orden ascendente. Aparece una flecha indicadora de orden en el encabezado.
4. Haga clic de nuevo en el mismo encabezado para invertir el orden a descendente.
5. Haga clic en un encabezado diferente para ordenar por esa columna, comenzando en orden ascendente.

## Consejos

- No todas las columnas se pueden ordenar. Si al hacer clic en un encabezado no ocurre nada, esa columna no admite ordenación.
- La columna **Frequency** utiliza ordenación numérica, por lo que 14.225 se ordena correctamente entre 14.200 y 14.300, en lugar de hacerlo como texto simple.
- La ordenación solo afecta al orden de visualización. No cambia el índice almacenado de ninguna memoria en la radio.
- Los filtros activos de búsqueda y perfil siguen vigentes durante la ordenación. Para ver todas las memorias en orden, borre el campo **Search:** y establezca **Profile:** en su valor predeterminado.
