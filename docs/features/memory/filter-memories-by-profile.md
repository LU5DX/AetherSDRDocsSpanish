# Filtrar memorias por perfil

Use el filtro de Perfil en el diálogo de Canales de Memoria para reducir la tabla de memorias a solo las entradas que pertenecen a un perfil global específico. Esto es útil cuando tiene una lista grande de memorias y desea ver solo los canales relevantes para su contexto operativo actual.

## Antes de comenzar

- La radio debe estar conectada. El diálogo de Canales de Memoria requiere una conexión activa con la radio.
- Debe existir al menos un perfil global en la radio. El cuadro combinado `Profile:` se completa con los perfiles globales activos de la radio.

## Pasos

1. Abra `Settings > Memory...`.
2. Localice el cuadro combinado `Profile:` en la fila de filtros en la parte superior del diálogo.
3. Haga clic en el cuadro combinado `Profile:` y seleccione el perfil por el que desea filtrar.
4. La tabla de memorias se actualiza inmediatamente para mostrar solo las entradas que coinciden con el perfil seleccionado.

Para limpiar el filtro y mostrar todas las memorias, seleccione la entrada en blanco o predeterminada en la parte superior del cuadro combinado `Profile:`.

## Consejos

- El filtro de perfil y la búsqueda por nombre (`Search:`) funcionan juntos. Puede seleccionar un perfil en `Profile:` y escribir en `Search:` para reducir aún más los resultados por nombre de memoria.
- El botón `Export...` respeta el filtro de perfil actual — solo se exportan las memorias visibles bajo el filtro activo.
- La barra de título sin marco y el redimensionamiento de bordes en 8 direcciones se pueden activar globalmente mediante la opción View → Frameless Window cuando se vuelve a abrir el diálogo de Canales de Memoria.
