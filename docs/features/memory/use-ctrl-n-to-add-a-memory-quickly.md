# Use Ctrl+N para agregar una memoria rápidamente

Agregue un canal de memoria desde la franja activa sin abrir ningún menú — solo presione un atajo de teclado.

## Antes de comenzar

- La radio debe estar conectada y tener una franja activa.
- El diálogo de Canales de Memoria no necesita estar abierto.

## Pasos

1. Presione **Ctrl+N** en cualquier lugar de la ventana principal de la aplicación.

   Se crea una nueva memoria a partir de la frecuencia, el modo y la configuración del filtro de la franja activa actual.

2. (Opcional) Abra **Settings > Memory...** para ver la nueva memoria en la tabla y editar su nombre u otros campos.

## Qué hace cada control

| Control | Comportamiento |
|---------|----------------|
| Atajo de teclado **Ctrl+N** | Crea una nueva memoria desde la franja activa — sin diálogo, sin confirmación. La memoria aparece en la tabla de Canales de Memoria. |

## Consejos

- Ctrl+N funciona incluso cuando otros diálogos tienen el foco, siempre que la ventana principal esté activa.
- Use **Settings > Memory...** para agregar, editar o eliminar memorias de forma masiva. Ctrl+N es el atajo más rápido para una sola memoria.
