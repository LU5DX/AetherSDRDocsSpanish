# Buscar memorias por nombre

Use el campo **Search:** en Memory Channels para filtrar la tabla y mostrar solo las memorias cuyo nombre coincida con lo que escriba. Esto es útil cuando tiene una gran cantidad de frecuencias almacenadas y desea encontrar una rápidamente.

## Antes de comenzar

- AetherSDR debe estar conectado al radio. Memory Channels requiere una conexión activa al radio.
- Abra Memory Channels mediante `Settings > Memory...`.

## Pasos

1. Abra `Settings > Memory...`.
2. Haga clic en el campo **Search:** en la parte superior del diálogo.
3. Escriba cualquier parte del nombre de la memoria. La tabla se filtra mientras escribe, no necesita presionar Enter para ver los resultados.
4. Para saltar y activar la fila actual, presione Enter.
5. Para borrar el filtro y mostrar todas las memorias, haga clic en el botón de limpieza dentro del campo **Search:**.

## Función de cada control

| Control | Comportamiento | Notas |
|---|---|---|
| **Search:** | Filtra la tabla de memorias por el nombre de la memoria. | Tiene un botón de limpieza. Presionar Enter activa la fila actual. |
| Tabla de memorias | Muestra solo las memorias cuyos nombres coinciden con el texto de búsqueda. | Combínelo con **Profile:** para reducir aún más los resultados. |

## Consejos

- El filtro se actualiza con cada tecla, por lo que puede dejar de escribir tan pronto como la lista sea lo suficientemente corta para encontrar su entrada.
- Después de filtrar, use Shift-clic o Ctrl-clic (Command-clic en macOS) para seleccionar varias filas de los resultados filtrados.
- Para buscar y sintonizar inmediatamente, escriba suficiente parte del nombre para aislar una fila, confirme que está resaltada y luego presione Enter.
