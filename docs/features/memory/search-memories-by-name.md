# Buscar memorias por nombre

Use el campo de Búsqueda en Memory Channels para filtrar la tabla y mostrar solo las memorias cuyo nombre coincida con lo que escriba. Esto es útil cuando tiene una gran cantidad de frecuencias almacenadas y desea encontrar una específica rápidamente.

## Antes de comenzar

- AetherSDR debe estar conectado a la radio. Memory Channels requiere una conexión activa con la radio.
- Abra Memory Channels mediante `Settings > Memory...`.

## Pasos

1. Abra `Settings > Memory...`.
2. Haga clic en el campo **Search:** en la parte superior del diálogo.
3. Escriba cualquier parte del nombre de la memoria. La tabla se filtra mientras escribe; no es necesario presionar Enter para ver los resultados.
4. Para saltar a la fila actual y activarla, presione Enter.
5. Para borrar el filtro y mostrar todas las memorias, haga clic en el botón de borrar dentro del campo **Search:**.

## Consejos

- El filtro se actualiza con cada pulsación de tecla, por lo que puede dejar de escribir tan pronto como la lista sea lo suficientemente corta para encontrar su entrada.
- Después de filtrar, use Shift-clic o Ctrl-clic (Command-clic en macOS) para seleccionar varias filas de los resultados filtrados.
- Para buscar y sintonizar inmediatamente, escriba suficiente del nombre para aislar una fila, confirme que esté resaltada y luego presione Enter.
- Use Ctrl+F para saltar rápidamente el foco al campo de búsqueda desde cualquier lugar del diálogo.
- Haga doble clic en la barra de título frameless para alternar maximizar/restaurar.
