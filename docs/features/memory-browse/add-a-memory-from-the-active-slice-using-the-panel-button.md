# Añadir una memoria desde el segmento activo usando el botón del panel

Guarde el segmento activo actual como una entrada de memoria en la lista de memorias del radio para poder recuperarlo más adelante.

## Antes de comenzar

- Debe haber un radio conectado.
- El panel del Explorador de Memorias debe estar visible en el panel lateral de la Ventana Principal.

## Pasos

1. Sintonice el segmento activo a la frecuencia y el modo que desee guardar.
2. En el panel del Explorador de Memorias, haga clic en **Add Memory**.

La frecuencia y el nombre del segmento se guardan como una nueva entrada de memoria, que aparece inmediatamente en la tabla de memorias.

## Función de cada control

| Control | Comportamiento |
|---------|---------------|
| **Add Memory** | Guarda el segmento actual como una nueva entrada de memoria. Siempre visible en la parte inferior del panel, anclado fuera de la tabla desplazable. Información sobre herramienta: "Guarde el segmento actual en este panadapter como una memoria". |

## Consejos

- La memoria se almacena en el propio radio, no localmente; el nombre y la frecuencia provienen de la configuración actual del segmento activo.
