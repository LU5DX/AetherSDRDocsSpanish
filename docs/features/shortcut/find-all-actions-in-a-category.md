# Encontrar todas las acciones de una categoría

El cuadro de diálogo de Atajos de teclado incluye un filtro de categoría que reduce la tabla de acciones a un solo grupo de acciones relacionadas. Úselo cuando desee revisar o editar todas las combinaciones de teclas de un área específica —por ejemplo, todas las acciones del VFO o todas las acciones del panadapter— sin tener que desplazarse por la lista completa.

## Antes de comenzar

- No se requiere conexión de radio.
- Abra el cuadro de diálogo mediante `View > Configure Shortcuts...`.

## Pasos

1. Abra `View > Configure Shortcuts...`.
2. En la fila de filtros sobre la tabla de acciones, localice el cuadro combinado **Category:**.
3. Haga clic en **Category:** y seleccione la categoría que desea ver.
4. La tabla de acciones se actualiza inmediatamente para mostrar solo las acciones de esa categoría, con las columnas **Action**, **Category**, **Current Key** y **Default Key**.
5. Para volver a la lista completa, seleccione **All** en **Category:**.

## Consejos

- Puede combinar **Category:** con el campo de texto **Filter:** para restringir aún más los resultados. Ambos filtros se aplican simultáneamente: la tabla muestra solo las filas que coinciden tanto con el texto como con la categoría seleccionada.
- El mapa de teclado sobre la tabla usa teclas codificadas por colores según la categoría. La fila de leyenda debajo del mapa identifica qué color pertenece a cada categoría, para que pueda localizar las teclas de una categoría de un vistazo antes de abrir el filtro.
