# Encontrar todas las acciones de una categoría

El diálogo Keyboard Shortcuts incluye un filtro de categoría que reduce la tabla de acciones a un único grupo de acciones relacionadas. Úselo cuando desee revisar o editar cada asignación de una sola área — por ejemplo, todas las acciones de VFO o todas las acciones del panadapter — sin desplazarse por la lista completa.

## Antes de comenzar

- No se requiere conexión a radio.
- Abra el diálogo mediante `View > Configure Shortcuts...`.

## Pasos

1. Abra `View > Configure Shortcuts...`.
2. En la fila de filtros ubicada sobre la tabla de acciones, localice el cuadro combinado **Category:**.
3. Haga clic en **Category:** y seleccione la categoría que desea ver.
4. La tabla de acciones se actualiza de inmediato para mostrar únicamente las acciones de esa categoría, con las columnas **Action**, **Category**, **Current Key** y **Default Key**.
5. Para volver a la lista completa, establezca **Category:** de nuevo en **All**.

## Consejos

- Puede combinar **Category:** con el campo de texto **Filter:** para reducir aún más los resultados. Ambos filtros se aplican de forma simultánea: la tabla muestra únicamente las filas que coinciden tanto con el texto como con la categoría seleccionada.
- El mapa de teclado ubicado sobre la tabla usa teclas codificadas por color según la categoría. La fila de leyenda que aparece debajo del mapa identifica qué color corresponde a cada categoría, lo que permite identificar las teclas de una categoría de un vistazo antes de abrir el filtro.
