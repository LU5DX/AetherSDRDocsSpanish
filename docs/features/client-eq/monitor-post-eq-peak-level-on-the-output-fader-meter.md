# Supervisar el nivel de pico posterior a la EQ en el medidor del Fader de Salida

El Fader de Salida en el editor flotante de EQ muestra un medidor de nivel de pico posterior a la EQ en vivo detrás de su control deslizante, lo que le permite ver cuánto margen de cabeza queda después de aplicar su curva de EQ y el ajuste de ganancia maestra.

## Antes de comenzar

- El editor flotante de EQ debe estar abierto para la ruta que desea supervisar (TX o RX). El Fader de Salida no está presente en el mosaico de la applet acoplado.
- La etapa de EQ debe estar habilitada. Si la etapa está puenteada, el medidor sigue reflejando la señal que pasa, pero la curva de EQ no tiene efecto.

## Pasos

1. Abra el editor flotante para la ruta que desea supervisar. Haga doble clic en la etapa de EQ correspondiente en el widget CHAIN — ya sea el lado TX o RX — para abrir el editor titulado "Aetherial Parametric EQ — TX" o "Aetherial Parametric EQ — RX".
2. Localice el Fader de Salida en el borde derecho del editor flotante. Es un fader y medidor de nivel combinados vertical.
3. Observe la barra de nivel detrás del control deslizante del fader. La barra muestra el nivel de pico posterior a la EQ suavizado en tiempo real usando un degradado verde-ámbar-rojo, igual que el medidor de nivel Tube.
4. Transmita o reciba audio con normalidad. La barra de nivel se actualiza continuamente, reflejando el nivel de la señal después de que se hayan aplicado todas las bandas de EQ y el desplazamiento de ganancia maestra.

El medidor de nivel detrás del control deslizante del fader es solo un indicador. Muestra el pico posterior a la EQ suavizado para la ruta vinculada a esta instancia del editor. No tiene un control separado ni una clave persistida.

## Consejos

- Si el medidor alcanza constantemente ámbar o rojo, arrastre el control deslizante del Fader de Salida hacia abajo para reducir la ganancia posterior a la EQ, o reduzca las ganancias de bandas individuales para bajar los niveles antes de la etapa maestra.
- Desplace la rueda del ratón sobre el Fader de Salida para ajustes finos de 0.5 dB mientras observa el medidor.
- Haga doble clic en el Fader de Salida para restablecerlo a 0 dB sin alterar sus ajustes de banda.
- Haga clic en la pantalla numérica en la parte inferior del fader para escribir un valor exacto en dB. El campo muestra un número simple cuando está enfocado; presione Enter o haga clic en otro lugar para confirmar. El valor se limita al rango de −36.0 a +12.0 dB.
