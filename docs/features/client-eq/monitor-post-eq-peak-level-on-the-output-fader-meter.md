# Monitor del nivel pico post-EQ en el medidor del control de nivel de salida

El control de nivel de salida (Output Fader) en el editor EQ flotante muestra un medidor de nivel pico post-EQ en tiempo real detrás de su manija, lo que le permite ver cuánto margen de pico queda después de aplicar su curva EQ y la configuración de ganancia maestra.

## Antes de comenzar

- El editor EQ flotante debe estar abierto para la ruta que desea monitorear (TX o RX). El control de nivel de salida no está presente en el mosaico de la miniaplicación acoplada.
- La etapa EQ debe estar habilitada. Si la etapa está en bypass, el medidor sigue reflejando la señal que pasa, pero la curva EQ no tiene efecto.

## Pasos

1. Abra el editor flotante para la ruta que desea monitorear. Haga doble clic en la etapa EQ correspondiente en el widget CHAIN, ya sea del lado TX o RX, para abrir el editor titulado "Aetherial Parametric EQ — TX" o "Aetherial Parametric EQ — RX".
2. Localice el control de nivel de salida (Output Fader) en el borde derecho del editor flotante. Es un control deslizante vertical combinado con un medidor de nivel.
3. Observe la barra de nivel detrás de la manija del control. La barra muestra el nivel pico post-EQ suavizado en tiempo real usando un gradiente verde-ámbar-rojo, igual que el medidor de nivel del tubo.
4. Transmita o reciba audio normalmente. La barra de nivel se actualiza continuamente, reflejando el nivel de la señal después de aplicar todas las bandas EQ y el desplazamiento de ganancia maestra.

El medidor de nivel detrás de la manija del control es solo un indicador. Muestra el pico post-EQ suavizado para la ruta vinculada a esta instancia del editor. No tiene un control separado ni una clave persistida.

## Consejos

- Si el medidor alcanza constantemente el ámbar o el rojo, arrastre la manija del control de nivel de salida (Output Fader) hacia abajo para reducir la ganancia post-EQ, o reduzca las ganancias de bandas individuales para bajar los niveles antes de la etapa maestra.
- Use la rueda del mouse sobre el control de nivel de salida para ajustes finos de 0.5 dB mientras observa el medidor.
- Haga doble clic en el control de nivel de salida para restablecerlo a 0 dB sin alterar la configuración de sus bandas.
