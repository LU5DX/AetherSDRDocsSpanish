# Eliminar una asignación

Elimine una única asignación MIDI de la tabla de asignaciones, o borre todas las asignaciones a la vez. Utilice esta función cuando una asignación de control ya no sea necesaria o desee reasignar un mando físico a un parámetro diferente.

## Antes de comenzar

- El cuadro de diálogo MIDI Controller Mapping debe estar abierto. Si no lo está, vaya a `Settings > MIDI Mapping...`.
- Debe tener al menos una asignación existente visible en la tabla de asignaciones.

## Pasos

### Eliminar una única asignación

1. Abra `Settings > MIDI Mapping...`.
2. En la tabla de asignaciones, localice la fila correspondiente a la asignación que desea eliminar.
3. Haga clic en `×` en la columna más a la derecha de esa fila.

La fila se elimina de inmediato. El cambio se guarda automáticamente.

### Eliminar todas las asignaciones a la vez

1. Abra `Settings > MIDI Mapping...`.
2. Haga clic en `Clear All`.

Se eliminan todas las filas de la tabla de asignaciones. El cambio se guarda automáticamente.

## Descripción de cada control

| Control           | Descripción                              | Notas |
|-------------------|------------------------------------------|-------|
| `×` (eliminar fila) | Elimina la única asignación de esa fila. |       |
| `Clear All`       | Elimina todas las asignaciones de la tabla. |       |

## Consejos

- Si elimina una asignación por error, puede restaurarla cargando un perfil guardado previamente. Consulte [Load a previously saved MIDI profile](load-a-previously-saved-midi-profile.md).
- Antes de utilizar `Clear All`, considere guardar sus asignaciones actuales como un perfil primero. Consulte [Save the current mapping as a named profile](save-the-current-mapping-as-a-named-profile.md).
