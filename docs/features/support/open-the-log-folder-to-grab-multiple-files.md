# Abrir la carpeta de registros para obtener varios archivos

Use esta página para abrir el directorio de registros en el explorador de archivos de su sistema operativo y así poder copiar, adjuntar o archivar varios archivos de registro a la vez.

## Antes de comenzar

- Abra AetherSDR.
- No se requiere conexión a una radio.

## Pasos

1. Haga clic en `Help > Support...` para abrir el diálogo Support & Diagnostics.
2. Haga clic en `Open Log Folder`.

El directorio de registros se abre en el explorador de archivos de su sistema operativo. Desde allí puede seleccionar, copiar o adjuntar los archivos según sea necesario.

## Qué hace cada control

| Control | Comportamiento |
|---|---|
| **Open Log Folder** | Abre el directorio de registros en el explorador de archivos del sistema operativo. |
| **Log path label** | Muestra la ruta completa al archivo de registro activo, para que sepa qué directorio se abrirá. |
| **Log file size** | Muestra el tamaño actual del archivo de registro activo. |

## Consejos

- La etiqueta de ruta de registro en el diálogo muestra el directorio exacto que abrirá `Open Log Folder`. Verifíquelo primero si no está seguro de dónde se almacenan los registros.
- Si desea capturar una reproducción limpia de un error, borre el registro antes de reproducir el problema y luego abra la carpeta para recuperar el archivo resultante.
