# Abrir la carpeta de registros para obtener varios archivos

Use esta página para abrir el directorio que contiene los archivos de registro de AetherSDR en el explorador de archivos de su sistema operativo, de modo que pueda copiar, adjuntar o archivar varios archivos de registro a la vez.

## Antes de comenzar

- No se requiere conexión de radio.
- Sepa dónde desea copiar los archivos (carpeta local, archivo adjunto de correo electrónico, carga en un sistema de seguimiento de incidencias).

## Pasos

1. Haga clic en `Help > Support...` para abrir el diálogo de Soporte y Diagnóstico.
2. Haga clic en `Open Log Folder`.

El explorador de archivos de su sistema operativo se abrirá directamente en el directorio de registros. Desde allí, seleccione y copie los archivos que necesite.

## Consejos

- La etiqueta de ruta del registro en la parte superior del diálogo muestra la ruta completa al archivo de registro actual, por lo que sabe qué directorio se abrirá antes de hacer clic.
- Si solo desea el archivo de registro actual en lugar de toda la carpeta, verifique la etiqueta de ruta del registro y navegue hasta allí manualmente.
- Para capturar un registro limpio antes de obtener los archivos, use `Clear Log` para truncar el archivo actual, reproduzca el problema y luego haga clic en `Open Log Folder`.
