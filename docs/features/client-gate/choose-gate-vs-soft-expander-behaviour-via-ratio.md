# Elegir comportamiento de gate o expansor suave mediante la relación

El control **Ratio** determina con qué agresividad el gate atenúa el audio por debajo del umbral. Configurar un valor bajo produce un expansor descendente suave que reduce gradualmente el audio silencioso; configurar un valor alto produce un gate duro que lo corta de forma brusca. Elegir la relación adecuada le permite adaptar el carácter del gate a su situación de ruido y estilo operativo.

## Antes de comenzar

- La etapa de gate debe estar habilitada en el lado que desea ajustar (TX o RX). Si el applet no es visible, habilite el gate a través del widget CHAIN o haga doble clic en la etapa GATE para abrir el editor flotante.
- Abra el subcontenedor **Aetherial TX Gate** (TX) o el subcontenedor **Aetherial AGC-T** (RX) dentro del contenedor principal Aetherial Audio (TXDSP) en el panel de applets.

## Pasos

1. Localice el control **Ratio** en la fila de cinco controles en la parte inferior del applet.
2. Para configurar el comportamiento de expansor suave, gire **Ratio** hacia un valor bajo (por ejemplo, 2.0:1). El audio por debajo del umbral se reduce gradualmente.
3. Para configurar el comportamiento de gate duro, gire **Ratio** hacia un valor alto (por ejemplo, 8.0:1 o superior). El audio por debajo del umbral se corta de forma brusca.
4. Observe la barra de reducción de ganancia mientras el audio pasa a través del gate. Una configuración de expansor suave produce un relleno ámbar más superficial y gradual; una configuración de gate duro produce un relleno profundo y abrupto cuando el gate se cierra.
5. Si el corte del gate duro es demasiado severo entre palabras, ajuste **Floor** para limitar la atenuación máxima. Consulte [Configurar Floor para evitar silencios artificiales entre palabras](set-floor-to-avoid-unnatural-silence-between-words.md).

## Consejos

- Una relación de 2.0:1 (el valor predeterminado) es un punto de partida conservador adecuado para la mayoría de los usos en TX. Auméntela solo si el ruido de nivel bajo sigue siendo audible cuando no está hablando.
- Con relaciones superiores a aproximadamente 8.0:1, el gate se comporta casi como un interruptor de encendido/apagado. Combínelo con un **Thresh** cuidadosamente ajustado para evitar recortar el borde inicial de las palabras.
- Use el control **Return** para eliminar el parloteo del gate. Si el gate parpadea abriéndose y cerrándose rápidamente cuando hace una pausa al hablar, aumente **Return** para que el gate permanezca abierto hasta que el nivel de entrada caiga bien por debajo del umbral. La banda cian en la curva de transferencia se amplía a medida que aumenta **Return**, mostrando directamente la zona de histéresis.
- La curva de transferencia se actualiza en tiempo real a medida que mueve **Ratio** o **Return**. Use la bola de entrada en tiempo real para confirmar que la forma de la curva y la banda de histéresis coinciden con su intención antes de transmitir.
- Los cambios en cualquier control toman efecto de inmediato y se guardan automáticamente. No se requiere ningún botón Apply ni Save.

## Solución de problemas

- **El control Ratio no tiene efecto en el sonido** — Confirme que la etapa de gate está habilitada. Un gate en modo bypass pasa el audio sin modificar, independientemente de la configuración de los controles. Consulte [Omitir el gate desde la cadena](bypass-the-gate-from-the-chain.md).
- **La relación del gate duro corta demasiado y crea silencios artificiales** — Reduzca **Floor** hacia 0 dB para disminuir la atenuación máxima, o reduzca **Ratio** hacia el rango del expansor suave.
- **La relación del expansor suave no suprime suficientemente el ruido** — Aumente **Ratio** o reduzca **Thresh** para que la atenuación comience a un nivel de entrada más alto.
- **El gate parlotea o parpadea en el umbral** — Aumente **Return** para que el gate permanezca abierto hasta que la señal caiga más por debajo del umbral. Observe cómo la banda de histéresis cian en la curva de transferencia se amplía al hacerlo.
