# Cambiar el nombre de un slot

Asigne un nombre significativo a un slot del DVK para identificarlo de un vistazo en lugar de usar la etiqueta predeterminada "Recording \<n\>".

## Antes de comenzar

- El panel Digital Voice Keyer debe estar visible en la ventana principal. Aparece cuando el DVK está habilitado o hay un modo de voz activo.
- Debe haber una conexión de radio establecida.

## Pasos

1. En el panel Digital Voice Keyer, localice el slot que desea renombrar (F1 a F8).
2. Haga clic derecho en la fila del slot.
3. En el menú contextual, seleccione "Rename".
4. La etiqueta del nombre del slot es reemplazada por el campo de texto "Rename edit". Escriba el nuevo nombre.
5. Presione Enter para confirmar, o presione Escape para cancelar sin guardar.

## Qué hace cada control

| Control | Comportamiento | Valor predeterminado |
|---|---|---|
| Etiquetas de nombre de slot | Muestra el nombre actual de cada slot. | `Recording <n>` |
| Rename edit | Campo de texto en línea que aparece cuando se activa un cambio de nombre a través del menú contextual. Escriba el nuevo nombre y presione Enter para aplicarlo. | — |

## Consejos

- Hacer doble clic en la etiqueta del nombre del slot también abre el campo "Rename edit" directamente.
- Presionar Escape mientras el campo "Rename edit" está abierto cancela el cambio de nombre y restaura el nombre anterior.
