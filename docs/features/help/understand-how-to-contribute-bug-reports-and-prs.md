# Entender cómo contribuir con informes de errores y PRs

Esta página explica cómo abrir la guía de contribución incluida en AetherSDR. La guía describe cómo registrar informes de errores y enviar pull requests sin necesidad de una conexión a internet.

## Antes de comenzar

- AetherSDR debe estar instalado. No se requiere conexión a un radio.

## Pasos

1. Haga clic en `Help` en la barra de menú.
2. Haga clic en `Contributing to AetherSDR...`.
3. Se abre el HelpDialog y muestra la guía de contribución en el visor de Markdown.
4. Para buscar dentro de la guía, escriba un término en el campo `Find:` y haga clic en `Next` para avanzar entre los resultados o en `Previous` para retroceder. Presione Return para ir al siguiente resultado. Presione Shift+Return para ir al resultado anterior.
5. Cuando termine, haga clic en `Close`.

## Qué hace cada control

| Control | Comportamiento |
|---|---|
| Campo `Find:` | Ingrese un tema o término para buscar dentro del tópico mostrado. El borde del campo se vuelve rojo cuando no hay coincidencias. |
| `Next` | Avanza a la siguiente coincidencia en el documento. Regresa al inicio cuando se llega al final. Se activa solo cuando el campo `Find:` no está vacío. |
| `Previous` | Retrocede a la coincidencia anterior en el documento. Regresa al final cuando se llega al inicio. Se activa solo cuando el campo `Find:` no está vacío. |
| `Close` | Cierra el diálogo. |

## Consejos

- El menú Help mantiene cada guía en un diálogo separado. Puede volver a abrir solo este tópico en cualquier momento sin afectar otras ventanas de ayuda abiertas.
- La guía de contribución está disponible sin conexión. No necesita una conexión de red para leerla.
