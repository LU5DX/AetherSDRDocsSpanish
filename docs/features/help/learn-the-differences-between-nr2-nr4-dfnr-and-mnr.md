# Aprenda las diferencias entre NR2, NR4, DFNR y MNR

Esta página explica cómo abrir la guía de cancelación de ruido incluida en AetherSDR, la cual describe las diferencias entre los modos de reducción de ruido NR2, NR4, DFNR y MNR disponibles en la FLEX-8600.

## Antes de comenzar

- No se requiere conexión de radio para leer este tema de ayuda.
- AetherSDR debe estar instalado con sus archivos de ayuda incluidos intactos.

## Pasos

1. Haga clic en `Help > Understanding Noise Cancellation...` en la barra de menú.
2. Se abre el HelpDialog con el título establecido en el tema de cancelación de ruido.
3. Lea el contenido en el visor Markdown. El tema cubre las diferencias entre NR2, NR4, DFNR y MNR.
4. Para buscar dentro del tema, escriba un término en el campo Find y haga clic en Next o Previous para desplazarse entre las coincidencias. Presione Return para encontrar la siguiente coincidencia; presione Shift+Return para encontrar la coincidencia anterior.
5. Haga clic en Close cuando termine.

## Consejos

- El borde del campo Find se vuelve rojo cuando no hay coincidencias para su término de búsqueda. Borre el campo o revise su término e intente nuevamente.
- Si el archivo de ayuda no se carga, AetherSDR muestra un mensaje en el visor. Reinstale la aplicación para restaurar los archivos de ayuda faltantes.
- Cada tema de ayuda en el menú Help es independiente. Puede volver a abrir solo este tema en cualquier momento sin alejarse de otras guías.
